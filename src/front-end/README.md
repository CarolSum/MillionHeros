# 去中心化益智比赛系统

使用truffle开发智能合约，将truffle compile得到的json文件放到contracts目录下，后面即可以用来生成合约实例。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 前端部署

运行`npm run build`之后，将生成的文件替换到根目录的src文件夹中
